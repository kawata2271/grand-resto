# GRAND RESTO

**やり込み型ファミレス経営シミュレーション**

「ザ・ファミレス」にインスパイアされた、リアル飲食店経営の全てを体験できるブラウザゲーム。開店しただけでは客は来ない。集客・清掃・仕込み・人事・設備管理——すべてを自分で判断して初めて店が回る、本格経営シムです。

## 遊び方

1. 最初は認知度が低く、ほとんど客が来ない
2. 集客施策（看板・チラシ・SNS等）で認知度を上げる
3. 客が来たら、スタッフをシフトに入れて営業
4. 閉店後は清掃・仕込み・発注を忘れずに
5. 稼いだお金でスタッフ雇用・設備投資・メニュー開発
6. 目標達成で「のれん分け」→新しいステージへ

## 主要機能

### 経営の深みを生むシステム
- **認知度システム**: 何もしないと毎日-0.3%。集客施策が命
- **清掃・衛生管理**: スタッフの体力を消耗する清掃 vs お金で業者委託。衛生スコアが低いと保健所で営業停止
- **仕込み・食材在庫**: 仕込み不足で品切れ、過多で廃棄のジレンマ
- **設備管理**: 故障すると調理不能。予防整備か壊れてから修理か
- **会計**: 月次P/L自動生成、融資申請、倒産判定

### スタッフ育成の奥深さ
- **6路線スキルツリー**: 技術/速度/創作（料理人）、接客/管理/PR（ホール）
- **特殊能力100種**: 採用時に40%で付与。伝説のシェフ〜破壊王まで
- **相性・師弟**: コンビ連携ボーナス、師弟関係で成長加速
- **労基法対応**: 週40h上限、残業代、有給休暇、新人OJT
- **体力管理**: 清掃や業務で消耗。体力切れで早退

### 周回要素
- **プレステージ（のれん分け）**: スタッフ・レシピ引き継ぎで新店舗
- **プロシージャル街マップ**: 毎回異なる街タイプで経営
- **チャレンジシナリオ**: 借金スタート/採用禁止/食材3倍
